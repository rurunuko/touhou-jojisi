うわばみ志向

June 15, 2017

新うわばみ志向についての雑感メモ

神霊廟追加パッチから、既存の無政府状態にならない旧うわばみ志向は
バニラと同じ「宗教志向」へと名称変更となります
そして、新たに追加される志向が

「新うわばみ志向（新は便宜上。ゲーム中はただの「うわばみ志向」）」

となります。
以下がその、新たに生まれ変わった新うわばみ志向の具体的性能です

志向名：うわばみ志向

・大醸造人発生時、4ターン（普通速度時）の黄金期を到来させる
　黄金期のターンは普通に偉人を黄金期として消費させた場合の半分のターン
　この黄金期にはマウソロス霊廟の効果が乗る
　酒主権一番乗りボーナスの大醸造人でも黄金期が発生する

・都市占領時のレジスタンス期間-50%

・国酒を採用した状態で新都市を建設した場合、新都市に無条件で国酒布教

・全都市ビーカー+10%

・生産速度2倍…天文台、各種酵母精製所（僧院）

まず目を引くのが大醸造人発生時の小黄金期ボーナス。
意図的に大醸造人を発生させ続ける事が出来れば、他志向使用時よりも
はるかに長い黄金期の恩恵を得る事が出来るだろう
上にも書いてあるが、この小黄金期はマウソロス霊廟の恩恵を受けれるため
普段あまり自分で建てる事はないであろうマウソロス霊廟だが、うわばみ志向に限っては
戦法によっては狙ってもいいかもしれない程度には価値が上昇している、といえる

また、統合MOD独自バランス調整で酒主権（神授王権）一番乗りで無償の大醸造人が貰えるが
この無償の大醸造人でも黄金期を発生させる事が出来る。
一番乗り出来た後は技術交換要員にしておけばいいので、狙ってみる価値はあるだろう

聖職者強化がある博麗神社、聖職者無限雇用が可能な白玉楼、
毘沙門天の寅で意図的にハゲを産出しまくることが出来る星蓮船と相性が良い
この辺の文明でハゲRを軸にした大うわばみ経済なんかはワンチャンあるだろう

次に特徴的なのは、国酒採用時に新都市建設時に無条件で国酒布教という点。
これは常に恩恵はあるだろうが、一番有難いのは序盤の拡張時に効力を発揮させることで
擬似創造志向状態にさせることが出来るということだろう
よって、うわばみ志向持ちに限り初期宗教創始の価値が上昇しているといっていい

他にも何も考えなくても恩恵を受けれるビーカー+10%やレジスタンス期間半減、
僧院や天文台の生産倍速などもついているため、特殊戦法に頼らなくても効力を発揮出来る
戦争屋にとってもレジスタンス半額の恩恵は大きいだろう

なお、この「新うわばみ志向」を作成する際に
CivFanatics内Downloads/Python Components内にある

Faithful Trait
