    * 早苗：帝国・哲学
    * 朱鷺子：労働・集権

    * 豊姫：拡張・宗教

    * サグメ：哲学・集権
    * レイセン2号：勤労・防衛

* ゲーム内部の処理を大幅に見直す

* 上記内部処理変更に関連し、社会制度変更時に自動建設される建造物（労働志向や信仰系など）は即時反映されるように

* 小町の独自志向等であった「再生成するたびに出力が増加するバグ」を修正

* その他、気付いた範囲にあった雑多なバグを修正

ただし内部データは消していないため、ランダム指導者時などは出てくる可能性あり

## 博麗神社

* 純狐の志向は白兵コスト-15%に変更
